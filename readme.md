## Installation

```sh
npm install -g speedracer
```

**Speed Racer** needs Chrome Canary to profile your files:

**Mac OS**

```sh
brew cask install google-chrome-canary
```

**Others**

Download [Chrome Canary](https://www.google.com/intl/en/chrome/browser/canary.html)
